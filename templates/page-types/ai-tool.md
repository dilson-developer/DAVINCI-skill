# Page Type — AI Tool / Dev Tool

Perfil: ferramenta técnica, API, ou produto de IA usado por developers ou equipas técnicas. Tom confiante, técnico mas acessível.

## Ordem de secções

1. `templates/sections/hero.md` — variante AI/dev tool, pode incluir termo técnico como prova de categoria
2. Faixa de prova social — logos de empresas ou menção a tecnologia/parceria reconhecida
3. `templates/sections/features.md` — tom tech-forward, pode usar humor leve
4. (Opcional) secção de aprofundamento com snippet de código ou demo interativa
5. `templates/sections/testimonials.md` — testemunhos técnicos ou métricas de performance
6. `templates/sections/pricing.md` — nomenclatura Pro/Enterprise é aceite neste tipo
7. `templates/sections/faq.md` — incluir pergunta técnica de integração/compatibilidade
8. `templates/sections/cta.md`

## Regras específicas deste tipo

- **Tipografia**: sans-serif geométrica + elemento mono para métricas/código [rules/typography.md #5]
- **Cor**: dark mode como padrão aceite, gradiente roxo/azul, acento neon único [rules/color-systems.md #6]
- **Copy**: termos de categoria (SDKs, API, MCP) usados como prova de credibilidade, não como explicação [rules/copywriting.md #6]
