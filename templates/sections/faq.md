# Template — FAQ

```
Secção: FAQ
- 4-6 perguntas, formato acordeão (expande ao clicar)
- Perguntas escritas na linguagem do utilizador, não como títulos de política [rules/copywriting.md #7]
- Respostas de 1-3 frases; para temas longos, linkar para documentação
```

## Perguntas que quase sempre devem estar presentes

1. Pergunta sobre preço/cancelamento ("Posso cancelar quando quiser?")
2. Pergunta sobre período de teste ("Preciso de cartão de crédito para experimentar?")
3. Pergunta sobre integração/compatibilidade com o que o utilizador já usa
4. Pergunta sobre suporte ("Como funciona o suporte?")

Ajustar as restantes 2 conforme as objeções específicas do produto.
