# Template — Testemunhos / Prova Social

```
Secção: Prova Social
- Se houver logos de clientes/parceiros: faixa monocromática logo após o hero [rules/color-systems.md #5]
- Se houver métricas: números concretos em destaque tipográfico grande (ex: "100+", "300%") com legenda curta abaixo
- Se houver testemunhos: 1-3 citações reais, cada uma com nome + cargo + empresa (nunca anónimo)
- Formato "antes/depois": usar quando o produto tem resultado comparável e mensurável (ver caso LinkedIn audit em references/design-patterns.md)
```

## Regra crítica

Nunca inventar números, logos ou testemunhos. Se o produto ainda não tem prova social real (MVP novo), usar em alternativa:
- Prova de credibilidade do fundador/equipa
- Garantia ou período de teste sem risco
- Prova de tecnologia/parceria (ex: "Construído com [tecnologia reconhecida]")
