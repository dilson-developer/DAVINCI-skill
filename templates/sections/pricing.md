# Template — Pricing

```
Secção: Pricing
- 2-3 planos, nunca mais (paralisia de escolha)
- 1 plano visualmente destacado como "recomendado" (borda/acento na cor de destaque)
- Cada plano: nome + preço + 1 frase de "para quem é" + lista de 4-6 itens + CTA próprio
- Preço em destaque tipográfico (peso 700, tamanho maior que o resto do card)
- Toggle mensal/anual se aplicável, com badge de desconto no anual ("-20%")
```

## Regras de conteúdo

- Nome do plano orientado a quem usa ("Solo", "Equipa", "Empresa"), não a features ("Basic", "Pro", "Enterprise") a menos que o público já espere essa nomenclatura (dev tools costumam aceitar Pro/Enterprise).
- Lista de itens: começar sempre pelo que o plano anterior já tinha ("Tudo do Solo, mais:") — reforça progressão em vez de repetir tudo.
- Se preço não estiver definido (contacto para vendas), usar "Fale connosco" como CTA, nunca deixar o card vazio.
