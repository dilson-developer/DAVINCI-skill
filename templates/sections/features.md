# Template — Features

```
Secção: Grid de Features
- Número de blocos: 3-5 (nunca mais) [rules/layout-grids.md #3]
- Cada bloco: ícone + título de 2-4 palavras orientado a benefício + 1 frase de apoio [rules/copywriting.md #4]
- Grid: 3 colunas desktop / 1 coluna mobile
- Gap entre cards: mínimo 24px
- Border-radius consistente com o resto da página [rules/layout-grids.md #6]
```

## Estrutura de cada bloco

```
{{icone}}
{{título_benefício}}  → ex: "Nunca fiques sem stock", não "Alertas automáticos"
{{frase_apoio}}       → o "como" técnico, 1 frase
```

## Quando usar secção de aprofundamento

Se 1-2 features são o verdadeiro diferenciador do produto, adicionar uma secção extra depois do grid com mais espaço, imagem/screenshot e 2-3 frases — não tentar caber tudo no grid inicial.
